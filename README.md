很多新手爸妈都会有类似的场景：

宝宝在垫子中间玩，你一边工作一边余光时不时瞄一眼

只要孩子爬远一点，你就会瞬间紧张

你想要的是：只要孩子离开“安全区”，系统自动提醒你

大部分人第一反应是：
这是不是得装摄像头 App、搞云端识别、连 NAS、配一堆复杂设备？

其实——完全不用。

这篇文章要分享的是一个超轻量的玩具级 Demo：

只需要一个网页，就能在浏览器里跑目标检测，实时盯着宝宝是不是还在画面中央的安全区。

我把它叫做：「宝宝看护电子围栏」。
